## [V1.2.1] - 2022-04-26 19:27

### 新增

- 新增事件： 被戳事件`可选择回复/回戳，回复可随机语句~`
- 使用方法：`插件目录下==>配置项.ini`新增以下配置

```javascript
[事件通知{开关}]
被戳事件=启用
[事件通知{通知}]
被戳事件=戳我干啥？[|]咋了？[|]有事就说别戳我！[|][回戳]
```
不像使用在开关那将`启用`改为`禁用`即可，通知处 `[|]` 为分割符！，如果其中用`[回戳]` 机器人就会戳回去，若其中有`[不通知]`就不会回复了（当然该回戳还是回戳！）
