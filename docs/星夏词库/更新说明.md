删除成功后 **右键** 选择 **执行SQL** 输入以下代码：
```sql
INSERT INTO "yz_order" ("分类", "指令", "权限", "成功回复", "失败回复", "参数个数", "参数位置", "开关") VALUES ('修改回复', '^改回复(.*)\s(\d*)\s(.*?)$', '90', '修改{id}回复成功', '修改{id}回复失败', 3, '{$1}|{$2}|{$3}', '开');
```
